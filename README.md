# がくしゅうアプリ(小学生向け)

100ます計算タイムアタックと、小学3年生の漢字れんしゅうができるWebアプリ。
サーバー不要・データはブラウザ内(localStorage)に保存されます。

**公開URL: https://notthi.github.io/gakushu-app/**

## 機能

### 🔢 100ますけいさん
- たし算/ひき算/かけ算、100ます・25ますを選択
- タイムアタック形式。全問正解のときだけ自己ベストとして記録
- 間違えたマスは赤く表示され、正解を確認できる
- スマホ:画面下のテンキーで入力/Chromebook:数字キー・Enter・Backspaceで入力

### ✏️ かんじれんしゅう(3年生・200字)
- 読み3択クイズ(1回10問)
- 間違えた字は「にがてちょう」に入り、出題されやすくなる
- 2回連続正解で「おぼえた」扱い。進捗ゲージで確認できる

### 🗾 ちずパズル(都道府県)
- 県の形をドラッグして白地図の正しい位置にはめるタイムアタック
- 「ぜんこく47」と地方ごと(北海道・東北/関東/中部/近畿/中国・四国/九州・沖縄)を選択
- 間違った場所に置く・ヒントを使うと+5秒のペナルティ
- 地図データ: [Geolonia japanese-prefectures](https://github.com/geolonia/japanese-prefectures)(MIT License)

### 🏆 きろく
- 種目別の自己ベスト、最近の履歴
- 練習した日にスタンプがつくカレンダー

## 使い方

- 上の公開URLをChromeで開く(Chromebook・スマホ共通)
- スマホはChromeのメニューから「ホーム画面に追加」でアプリのように使える
- ローカルで試す場合は `index.html` をChromeで開くだけでも動く

## ファイル構成

| ファイル | 内容 |
|---|---|
| `index.html` | 画面(ホーム/100ます/漢字/きろく) |
| `style.css` | デザイン |
| `app.js` | 画面切り替え・保存・音・きろく画面 |
| `hyakumasu.js` | 100ます計算のロジック |
| `kanji.js` | 漢字クイズのロジック |
| `kanji_data_g3.js` | 小3配当漢字200字のデータ |
| `map.js` | 都道府県パズルのロジック |
| `japan_map_data.js` | 47都道府県の形データ(Geolonia製SVGから変換) |
