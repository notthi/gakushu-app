# がくしゅうアプリ(小学生向け)

100ます計算タイムアタック、漢字れんしゅう(1〜3年生)、都道府県パズル、ローマ字タイピング、英検5級対策ができるWebアプリ。
サーバー不要・データはブラウザ内(localStorage)に保存されます。

**公開URL: https://notthi.github.io/gakushu-app/**

## 機能

### 🔢 100ますけいさん
- たし算/ひき算/かけ算、100ます・25ますを選択
- タイムアタック形式。全問正解のときだけ自己ベストとして記録
- 間違えたマスは赤く表示され、正解を確認できる
- スマホ:画面下のテンキーで入力/Chromebook:数字キー・Enter・Backspaceで入力

### ✏️ かんじれんしゅう(1〜3年生)
- 1年生(80字)/2年生(160字)/3年生(200字)を切り替え可能。前回選んだ学年を記憶
- 読み3択クイズ(1回10問)
- 間違えた字は「にがてちょう」に入り、出題されやすくなる
- 2回連続正解で「おぼえた」扱い。学年ごとに進捗ゲージで確認できる

### 🗾 ちずパズル(都道府県)
- 県の形をドラッグして白地図の正しい位置にはめるタイムアタック
- 「ぜんこく47」と地方ごと(北海道・東北/関東/中部/近畿/中国・四国/九州・沖縄)を選択
- 間違った場所に置く・ヒントを使うと+5秒のペナルティ
- 地図データ: [Geolonia japanese-prefectures](https://github.com/geolonia/japanese-prefectures)(MIT License)

### ⌨️ ローマじタイピング
- ひらがなの単語をローマ字で打つタイムアタック(1回10問)
- 「かんたん」(基本の音のみ)と「ふつう」(ん・っ・ゃゅょ入り)の2コース
- 訓令式(si・ti・tu)を基本表示しつつ、ヘボン式(shi・chi・tsu)の入力も受け付ける
- Chromebookは物理キーボード、スマホは画面上のキーボードで入力

### 🔤 えいご(英検5級)
- たんごクイズ(4択・10問):英検5級レベルの単語 約340語。「えいご→いみ」「いみ→えいご」の両方向に対応
- あなうめ もんだい(4択・10問):英検の大問1・2ふうの文空所補充(be動詞・一般動詞・can・疑問詞・前置詞・会話表現)
- ならべかえ もんだい(5問):日本文に合わせて単語をタップして英文を作る(英検の大問3ふう)
- 間違えた単語は「にがてちょう」に入り、出題されやすくなる。2回連続正解で「おぼえた」扱い
- 出題時・解答時に英語を読み上げ(Web Speech API。音OFFなら読み上げなし)

### ⏱️ タイマー
- 3分固定のタイマー。カウントダウン/カウントアップを選択できる
- 1秒ごとに残り(または経過)秒数を英語で読み上げ(one, two, three…one hundred eighty)
- 開始時は「Start!」、終了時は「Time's up!」と読み上げてファンファーレが鳴る

### 🏅 ランキング(名前つき記録)
- 記録を出した後(100ますは全問正解時のみ)、名前を選んで登録できる
- 名前は新規入力 or 過去に使った名前から選択(最大8人ぶん記憶)
- 種目・モードごとにトップ10を表示。きょうだいやおうちの人と競える

### 🏆 きろく
- 種目別の自己ベスト、最近の履歴
- 練習した日にスタンプがつくカレンダー

## 使い方

- 上の公開URLをChromeで開く(Chromebook・スマホ共通)
- スマホはChromeのメニューから「ホーム画面に追加」でアプリのように使える
- ローカルで試す場合は `index.html` をChromeで開くだけでも動く

## ファイル構成

| ファイル | 内容 |
|---|---|
| `index.html` | 画面(ホーム/100ます/漢字/きろく) |
| `style.css` | デザイン |
| `app.js` | 画面切り替え・保存・音・きろく画面 |
| `hyakumasu.js` | 100ます計算のロジック |
| `kanji.js` | 漢字クイズのロジック(学年切り替え対応) |
| `kanji_data_g1.js` | 小1配当漢字80字のデータ |
| `kanji_data_g2.js` | 小2配当漢字160字のデータ |
| `kanji_data_g3.js` | 小3配当漢字200字のデータ |
| `map.js` | 都道府県パズルのロジック |
| `japan_map_data.js` | 47都道府県の形データ(Geolonia製SVGから変換) |
| `typing.js` | ローマ字タイピングのロジック(判定エンジン含む) |
| `typing_words.js` | タイピングの単語リスト |
| `eiken.js` | 英検5級クイズのロジック(たんご/あなうめ/ならべかえ) |
| `eiken_data.js` | 英検5級の単語(約340語)・あなうめ44問・ならべかえ24問のデータ |
| `timer.js` | 1秒読み上げタイマーのロジック(3分固定・カウントアップ/ダウン) |

## 補足:キャッシュ対策

JS/データファイルの `<script>` タグには `?v=2` のようなバージョンクエリを付けています。
中身を変更したときはこの数字を上げないと、スマホ・Chromebookのブラウザキャッシュが古いファイルを使い続けることがあります。
